# Branching Show Commands

Display information about your Git repository or Repo workspace.

## `sc show branch`

### Behaviour

- Git: Shows branch information for your git repository.
- Repo: Shows branch information, lock status and groups of all projects.

## `sc show repo-flow-config`

### Behaviour

- Git: Not applicable, errors.
- Repo: Show git-flow config for all projects.

## `sc show log`

### Behaviour

- Git: Shows information about most recent commit on your git repository.
- Repo: Shows information about most recent commit on all projects.

## `sc show tag <tag>` or `sc tag show <tag>`

### Behaviour

- Git: Shows information about the particular tag in your git repository.
- Repo: Shows information about the particular tag in all projects.
